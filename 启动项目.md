# 航小票 - 项目启动指南

## 📋 项目概述

航小票是一个面向沈阳航空航天大学的校园活动抢票小程序。

**技术栈**：
- **后端**：NestJS + TypeORM + SQLite
- **前端**：微信小程序原生开发
- **数据库**：SQLite（文件形式）

---

## 🚀 第一步：环境准备

### 1.1 安装必要软件

请确保已安装以下软件：

#### Node.js（必需）
- 下载地址：https://nodejs.org/
- 版本要求：Node.js 16+ 和 npm 8+
- 验证安装：`node --version` 和 `npm --version`

#### 微信开发者工具（必需）
- 下载地址：https://developers.weixin.qq.com/miniprogram/dev/beta.html
- 选择 Windows 稳定版下载并安装

#### Git（可选但推荐）
- 下载地址：https://git-scm.com/download/win
- 用于版本管理

## 🚀 第二步：启动后端服务

### 2.1 安装后端依赖

打开终端（PowerShell 或命令提示符）：

```bash
cd D:\Users\Lenovo\Desktop\hangxiaopiao\backend
npm install
```

等待安装完成（约1-2分钟）。

### 2.2 启动后端服务

```bash
npm run start:dev
```

你应该看到类似以下输出：

```
[Nest] 服务ID - 进程ID
 LOG [InstanceLoader] TypeOrmModule dependencies initialized +35ms
 LOG [InstanceLoader] ActivitiesModule dependencies initialized +35ms
 LOG [InstanceLoader] TicketsModule dependencies initialized +1ms
 LOG [RoutesResolver] ActivitiesController {/activities}: +4ms
 LOG [NestApplication] Nest application successfully started +2ms
```

### 2.3 验证后端服务

打开浏览器，访问：http://localhost:3000/activities

如果看到 JSON 数据输出，说明后端服务启动成功！

### ⚠️ 2.4 常见问题

**问题1：端口被占用**
```
Error: listen EADDRINUSE :::3000
```
**解决方案**：
1. 打开任务管理器
2. 结束占用端口 3000 的进程
3. 或者修改 `.env` 文件中的端口：`PORT=3001`

**问题2：缺少依赖**
```
npm install 失败
```
**解决方案**：
```bash
# 清除缓存后重试
npm cache clean --force
npm install
```

1. 打开微信开发者工具

## 🗄️ 第四步：查看和管理数据库

### 4.1 数据库文件位置

数据库文件位于：`D:\Users\Lenovo\Desktop\hangxiaopiao\backend\database. db`

这是一个 SQLite 数据库文件，可以使用多种工具查看。

### 4.2 使用 SQLite Viewer（推荐）

**在线查看器**：
1. 访问 https://inloop.github.io/sqlite-viewer/
2. 将 `database. db` 文件拖入页面
3. 即可查看所有表和数据

**桌面应用**：
- 下载 DB Browser for SQLite：https://sqlitebrowser.org/
- 安装后打开 `database. db` 文件即可

### 4.3 使用命令行查看

如果安装了 SQLite 命令行工具：

```bash
cd D:\Users\Lenovo\Desktop\hangxiaopiao\backend

# 进入 SQLite
sqlite3 database.db

# 查看所有表
sqlite> .tables

# 查看用户表内容
sqlite> SELECT * FROM users;

# 查看活动表内容
sqlite> SELECT * FROM activities;

# 退出
sqlite> .quit
```

### 4.4 通过 API 查看数据

#### 查看所有用户：
```bash
curl http://localhost:3000/users
```

#### 查看所有活动：
```bash
curl http://localhost:3000/activities
```

#### 查看特定用户：
```bash
curl http://localhost:3000/users/20230101
```

## 📱 第五步：功能流程

### 5.1 首次使用流程

1. **打开小程序** → 显示注册页面
2. **填写信息**：
   - 姓名：你的真实姓名
   - 学号：8-12位数字
   - 学院：从下拉菜单选择
   - 班级：如"2023级1班"
   - 手机号：11位数字
3. **点击"完成注册"** → 跳转到首页
4. **注册信息**已保存到数据库

### 5.2 抢票流程

1. **首页/抢票** → 查看活动列表
2. **点击活动** → 查看活动详情
3. **点击"立即抢票"** → 抢票成功
4. **查看票根** → 在"我的"页面查看

### 5.3 编辑资料

1. **"我的"页面** → 点击"编辑资料"
2. **修改信息** → 点击"保存"
3. **数据更新** → 数据库同步更新

## 🔧 第六步：常见问题解决

### 问题1：前端无法连接后端

**症状**：小程序加载失败或显示空白

**解决方案**：
1. 确保后端服务正在运行（看到 `Nest application successfully started`）
2. 在微信开发者工具中勾选"不校验合法域名"
3. 重启后端服务

### 问题2：注册失败

**症状**：点击"完成注册"没有反应或显示错误

**解决方案**：
1. 检查后端服务是否正常运行
2. 确认表单填写是否完整（所有必填项）
3. 检查学号是否已被注册

### 问题3：数据库文件不存在

**症状**：提示找不到 `database. db`

**解决方案**：
1. 后端服务会在首次运行时自动创建数据库文件
2. 确保后端服务已成功启动
3. 检查路径：`D:\Users\Lenovo\Desktop\hangxiaopiao\backend\`

### 问题4：编译错误

**症状**：WXML 文件编译错误

**解决方案**：
1. 清除缓存：工具 → 清除缓存 → 清除全部
2. 重新编译：`Ctrl + Shift + P` → 输入 `build`

## 📊 数据库表结构

### users（用户表）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| name | VARCHAR | 姓名 |
| studentId | VARCHAR | 学号（唯一） |
| college | VARCHAR | 学院 |
| className | VARCHAR | 班级 |
| phone | VARCHAR | 手机号 |
| openid | VARCHAR | 微信openid（可选） |
| createdAt | DATE | 创建时间 |
| updatedAt | DATE | 更新时间 |

### activities（活动表）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| title | VARCHAR | 活动标题 |
| description | TEXT | 活动描述 |
| totalTickets | INTEGER | 总票数 |
| availableTickets | INTEGER | 剩余票数 |
| startTime | DATE | 开始时间 |
| endTime | DATE | 结束时间 |
| status | VARCHAR | 状态 |
| createdAt | DATE | 创建时间 |
| updatedAt | DATE | 更新时间 |

### tickets（门票表）
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INTEGER | 主键，自增 |
| activityId | INTEGER | 关联活动ID |
| userId | INTEGER | 关联用户ID |
| ticketNumber | VARCHAR | 票号（唯一） |
| seatNumber | VARCHAR | 座位号 |
| status | VARCHAR | 状态 |
| createdAt | DATE | 创建时间 |
